# Pictures-Rename

## Description

Pictures Rename is a python script that allows to rename all the files of a specific folder using a template name.

## Usage

Command line :
```bash
$ python3 pictures-rename.py src_dir newname newtype
```

## Examples :

Rename all the files in Test folder by IMG_i.jpg with i the number of the element
```bash
$ python3 pictures-rename.py /Desktop/Test IMG_ jpg 
```

Rename all the files in Test folder by July_i.png with i the number of the element
```bash
$ python3 pictures-rename.py /Desktop/Test July_ png 
```
